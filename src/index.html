Createthirdplaces.com is a website created by Bharat Ponnaluri to promote accessible public spaces. It is inspired by the organization Strong Towns, which empowers people to improve towns and neighborhoods through bottom up change. Createthirdplaces.com is a non-commercial open source entity.

Current projects

  • Places.js framework for building websites promoting in person interaction
  • dmvboardgames.com: The goal of dmvboardgames.com is to promote human connection through board games as a way of increasing interest in public spaces. It uses the places.js framework.
